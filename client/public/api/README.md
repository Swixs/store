# Скрипт сохранения пользователей (JSON)

Пользователи сохраняются в **users.json** на сервере — можно зарегистрироваться на одном ПК и войти с другого.

## Как запустить

1. **React** (один терминал):
   ```
   npm start
   ```

2. **PHP** (второй терминал), из папки проекта:
   ```
   cd public
   php -S localhost:8080
   ```

В `package.json` указан `"proxy": "http://localhost:8080"` — запросы к `/api/auth.php` уходят на PHP, данные пишутся в `public/api/users.json`.

## На хостинге

Залить на сервер с PHP папку `build` и папку `public` (с содержимым `api/`). Тогда `/api/auth.php` и `users.json` будут на том же домене.
